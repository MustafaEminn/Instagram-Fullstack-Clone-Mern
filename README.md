## Usage

### Clone
```bash
git clone https://github.com/MustafaEminn/Instagram-Fullstack-Clone-Mern
```

### Client
```bash
cd client
npm install & yarn install
npm start & yarn start
```

### Server
```bash
cd server
npm install & yarn install
npm start & yarn start
```
